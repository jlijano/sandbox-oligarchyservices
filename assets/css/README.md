# CSS assets

Place shared stylesheets here for the public site.

Current compatibility note: the live pages still load the existing flat `/assets/*.css` files. New CSS work should be added here first, then the public page references can be migrated in small visual-checked steps.

Recommended layout:

- `styles.css`: global tokens, layout, shared components, forms, and page sections.
- `hero-reference.css`: homepage hero media and type treatment.
- `process-infographic.css`: homepage process workflow styling.
- `footer.css`: refined footer layout and responsive footer rules.
