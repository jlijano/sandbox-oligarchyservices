# Image assets

Place source-controlled image assets here.

Use this folder for logos, icons, screenshots, exported graphics, compressed photos, and other bitmap or SVG image files that belong to the site.

Current notes:

- The homepage hero still uses an external Unsplash poster and `/assets/hero-loop.mp4` for video.
- Inline data-URI logo artwork still exists in current CSS/JS compatibility files.
- When adding new image files, prefer descriptive names such as `oligarchy-logo-red.png` or `homepage-hero-poster.jpg`.
